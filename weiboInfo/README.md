## 项目简介
这是一个通过python requests库获取新浪微博博文并且生成词云的脚本。

## 运行环境
python 3.5
windows 10 64bit

## 项目结构/文件用法

1. weibo.py - 主程序
2. test*.py - 测试用的参考代码
3. *.json   - 抓取到的JSON文件，可以导入文本编辑器查看结构


## 运行方法
`python weibo.py`
